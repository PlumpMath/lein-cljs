# lein-cljs

A minimal lein plugin to use the ClojureScript compiler

## Usage

    lein cljs {watch-mode} {build-id}

Where `watch-mode` is either `auto` or `once` and `build-id` is any of
the ids under the `:cljs` map in your `project.clj`.
